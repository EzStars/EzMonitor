# **EzMonitor** **SDK**

前端监控 SDK

## **功能特性**

-   **性能监控**：获取页面加载性能数据；

-   **错误监控**：捕获和上报 JavaScript 错误、资源加载错误、Promise 未捕获异常、集成到vue、react框架上；

-   **用户行为分析**：记录用户交互数据，为优化提供依据；

-   **异常监控**：捕获和上报页面异常情况

-   **数据上报**：合理的对数据进行上报。

## **安装**

已经发布，下面这行命令可以安装

`npm i @EzStars/EzMonitor`

## **环境测试**

```
  pnpm run build
  node test/test.js
  
```
